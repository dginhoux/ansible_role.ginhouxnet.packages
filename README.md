ginhouxnet.packages
=========

This ansible role manage linux packages.
It can be used for : 
* install and remove packages
* check for update
* upgrades packages
* clean cache
* autoremove


Requirements
------------

This role is built to only run on platforms defined in `meta/main.yml`


Role Variables
--------------

Necessary variables are defined on `defaults/main.yml`



Dependencies
------------
